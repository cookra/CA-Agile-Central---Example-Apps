KanbanScheduleState
=========================


## Overview
This app shows a grouped list of Artifacts (Stories, Defects), that match the following query condition:

((KanbanScheduleState = "") OR ((KanbanScheduleState != ScheduleState) AND (KanbanScheduleState != "Validation")))
